# Example datasets

So far we have uploaded datasets for a [MACS-cleared mouse lung](https://doi.org/10.5281/zenodo.20489678) and a [REPLICA-cleared mouse kidney](https://doi.org/10.5281/zenodo.20490386) to Zenodo.
These datasets include both raw and deskewed data, along with a maximum intensity projection (MIP).
These MIPs are shown at 1080p resolution below (click on the image to view full size).
